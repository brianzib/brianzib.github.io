Fundamentals Of Computer Systems 

 
	

	A Computer consists of many different parts, physical and intellectual. These parts are classified into either  Hardware 
	or  Software . Hardware refers to the physical components that make up the computer, e.g. RAM. While software refers 
	to the instructions or more generically: the programs that the computer can run. 
    

	
 Classification of Software 

	

	There are two types of software that have two completly different roles. There is  System Software  and  
	Application Software . 
	
	System Software refers to the algorithims necessary for the dunctioning of the computer. In other words the computer cannot function
	withought it. An example of System Software is the  Operating System .
	
	Application Software is completly different. This form of software is a window for the user to be able to interact wwith the computer
	and therefore complete tasks. 

	
	
 System Software 

	

	System software as stated above is the algorithims utilized to acctually make the computer work.
	Withought these the computer would not be able to function at all. There are several diffrernt types of 
	system software. 
	
  •  Operating Systems 
  •  Utility programs 
  •  Libraries 
  •  Translators (complier, assembler, interpreter) 

	

	The operating system provides an interface between the user and the Hardware. This is not acurate as the user 
    does not acctually directly interact with the hardware, but rather via the application software. 
	operating system also comes with a number of utility programs like:
	

	
  •  Encryption Software 
  •  COmpression Software 
  •  Defragmentation Software 

	
	
	
 Logic Gates and Boolean Algebra 

	
	Logic gates are ciruits that take an input and provide an output based on a certain type
	of logical condition. 
	There are different types of Logic gates:
	
    
	
  •  AND gate. It takes two inputs (either 1 or 0). In order for it to return a true value (1) both 
	Input values have to be true to begin with
    • 
	
	 
	
	
  •  OR gate. It takes two inputs (either 1 or 0). in order for it to return a true value
	one of the input value must be true 
    • 
	
	 
	
	
  •  NOT gate. It only takes 1 value as an input, and flips whatever than value is
	(so 1 = 0 and 0 = 1)
    • 
	
	 	
	
	
  •  NOR gate. It takes two inputs (either 1 or 0). It only outputs a true value if  BOTH 
	values are negative 
    • 
	
	 
	
	
  •  XOR gate. It takes two inputs (either 1 or 0). In order for it to return a true value
	One of the two values must be true, but only one or else it returns a fales (0) value
    • 
	
	 
	
	
  •  NAND gate. It takes two inputs (either 1 or 0). It outputs a true value when there is a false value
	as an input	
    • 
	
	 
	

	
	

	These logic circuits can be combined to produce different results based on certain conditions. It is also possible to do 
	arithmatic functions with these circuits. In the specification only  half and full adders  are covered. 
	

	
	
 Half and Full bit adders 

	

	These are circuits that utilize boolean login in order to complete addition. A half adder adds two bits together giving answer and a carry out. 
	A Full-adder adds two bits together and takes into account the carry in form the last addition, aswell as giving out a carry and a data bit.
